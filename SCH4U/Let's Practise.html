Q. Determine the electronegativity difference, and classify the type of bond in the following pairs of atoms:

a. N--H

b.  P--H

c.  K--Br

Show Answer

Solution:

Step 1: Using the electronegativity values shown earlier, predict the difference in electronegativity between atoms.

ΔENK-Br  = ENK - ENBr

ΔENK-Br  = 2.8 - 0.8

ΔENK-Br = 2.0

Step 2: Using the table of electronegativity difference and bond types, classify the type of bond in each compound.

a. N--H

ΔENN-H = 0.9

Since 0.5 < ΔENN-H < 1.7, N--H is a polar covalent bond.

b. P--H

ΔENP-H =0

Since ΔENP-H < 0.5, P--H is a nonpolar covalent bond.

c. K-Br

ΔENK-Br  = 2.0

Since ΔENK-Br > 1.7, K-Br is an ionic bond.
