Learning Goals

Students will be able to:

Learn about different types of bonds.

Understand how to draw Lewis structures of compounds.

Learn about electronegativity.

Learn about bond polarity.

Be able to predict the electronegativity difference.

Be able to classify the type of bond in a pair of atoms.

Learn about the polarity of a molecule.
