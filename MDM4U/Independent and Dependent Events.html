Independent Events

Sometimes the probability of one event depends on whether another event occurs, and sometimes it does not. When one event has no effect on the probability of another, the events are independent.

Conditional Probability

When the outcome of one trial has been determined, then the conditional probability of a subsequent trial can be calculated based on the result of the first trial. If the first event is A and the second event is B, then P(B|A) represents the conditional probability that B will occur, given that A has occurred. To calculate the probability of both dependent events occurring, apply the multiplicative principle for dependent events.
