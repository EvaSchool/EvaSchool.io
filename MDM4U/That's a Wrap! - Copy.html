That's a Wrap!

Learning Goals

  • Make lists, charts, and tree diagrams to organize counting.
  • Use the fundamental counting principle for counting and solving problems.
  • See how using permutations has advantages over other counting techniques.
  • Solve simple problems using techniques for counting permutations.
  • Write permutation solutions using proper mathematical notation.
  • Use the rule of sum to solve counting problems.
  • Solve probability problems using counting principles for situations with equally likely outcomes.

Success Criteria

  • You will know that you are successful if you can:
    Make a tree diagram to solve a counting probability problem.
  • Describe the fundamental counting principle (m and n = m x n).
  • Describe what the term “factorial” means and recognize its symbol.
  • Explain that the number of permutations of n items is n factorial.
  • Explain how to calculate r permutations of n.
  • Describe the rule of sum.
  • Explain when to use the rule of sum.
  • Describe the likeliness of an event occurring using probability.
  • Calculate simple probability.
