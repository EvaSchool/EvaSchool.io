That's a Wrap!

Learning Goals

  • Use probability to describe the likelihood of something occurring.
  • Measure and calculate simple probabilities.
  • Calculate theoretical probability using sample spaces and tree diagrams.
  • Compare probability and odds.
  • Recognize the difference between experimental probability and theoretical probability.
  • Describe how an event can represent a set of probability outcomes.
  • Calculate the probability of an event occurring.
  • Connect the probabilities of different events within the same situation.
  • Solve probability problems involving multiple events.

Success Criteria

You will know that you are successful if you can:

  • Describe the likeliness of an event occurring using probability.
  • Calculate simple probability.
  • Draw a tree diagram to help calculate the probability of an event occurring.
  • Define probability and odds.
  • Tell whether an event represents experimental probability or theoretical probability, and explain why.
  • Relate an event to probability outcomes.
  • Calculate the probability of an event occurring.
  • Explain how to connect the probabilities of different events within the same situation.
  • Explain how to solve probability problems involving multiple events.
