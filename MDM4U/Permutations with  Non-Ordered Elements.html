Permutations with  Non-Ordered Elements

Definition of Terms

Permutations is a mathematical technique that determines the number of possible  arrangements.

For example, some permutations of the letters ABCDE are:
         AEBCD           BCDEA       CBEAD          DEABC
How many such permutations are possible? Since there are five choices for the first position, four for the second(after the first has been chosen), three for the third ( after first two have been chosen) and so on. The fundamental Counting Principle 
Tells us that the number of possible permutation is: 5 × 4 × 3 × 2 × 1 = 120

How many permutations consisting of three letters can be made from these five letters (ABCDE)? Some of these permutations are:

ABC         BDE          ABE          CDB         BED

Again, there are five choices for the first position, four for the second, three for the third. By the Fundamental Counting Principle the number such permutation is 

5 × 4 × 3 = 60

In general, if a set has n elements, then the number of ways of ordering r elements from the set is denoted by P(n,r) and is called the number of permutations of n objects taken r at a time.
