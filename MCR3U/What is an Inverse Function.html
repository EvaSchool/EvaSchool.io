What is an Inverse Function?

 

Callout

The inverse of a function is the set of ordered pairs obtained by interchanging the first and second elements of each pair in the original function. In other words, finding an inverse is simply the swapping of the x and y coordinates.

Example 1: f (x) = {(2,3), (4,5), (-2,6), (1,-5)}

Answer: The inverse of f (x) = {(3,2), (5,4), (6,-2), (-5,1)} (It's a function)

Example 1: g (x) = {(4,1), (8,3), (-5,3), (0,1)}

Answer: The inverse of  g (x) = {(1,4), (3,8), (3,-5), (1,0)}   (NOT a function, x's repeat)
