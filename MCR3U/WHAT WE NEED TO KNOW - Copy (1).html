What We Need To Know

PERIODIC FUNCTION  

a function whose graph repeats at regular intervals; the y-values in the table of values show a repetitive pattern when the x-values change by the same increment.

PERIOD 

the change in the independent variable (typically x) corresponding to one cycle; a cycle of a periodic function is a portion of the graph that repeats.

Callout

PEAK

maximum point on the graph

TROUGH

the minimum point on the graph
