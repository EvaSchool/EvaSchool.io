Q 1. Use the vertical line test to determine which of the following is a function. Explain. 

Show Answer

The vertical line test is a graphical method used to determine if a given graph represents a function. In this test, you draw vertical lines on the graph. If any vertical line intersects the graph at more than one point, then the graph does not represent a function. If every vertical line intersects the graph at most once, then it represents a valid function.  So graph (a) and (b) is a function.
