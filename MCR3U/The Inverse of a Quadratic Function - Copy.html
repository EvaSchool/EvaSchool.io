The Inverse of a Quadratic Function 

Key Idea

  • The inverse of the original function undoes what the original function has done. It can be used to determine which values of the original dependent variable produce given values for the original independent variable.
  • The inverse of a quadratic function defined over all the real numbers is not a function. It is a parabolic relation that opens either to the left or to the right. If the original quadratic opens up (a>0)  the inverse opens to the right. If the original quadratic opens down (a<0) the inverse opens to the left.
