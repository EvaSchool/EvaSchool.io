Lesson 4.2: Media Literacy - Content

LITERACY

  • Traditionally, it means the ability to read and write.
  • Over the years, the meaning of literacy has changed.
  • It now means competence in proper communication.

Callout

COMBINE

MEDIA + LITERACY

What do you think it means?

 

MEDIA LITERACY

  • The ability to read and interpret media messages critically.
  • This also means that we can create appropriate media messages.

Callout

Why is Media Literacy important?

 

Media literacy is an essential skill for anyone living in modern times. This is a weapon against disinformation and propaganda.

To learn more about media literacy, watch this video. 

 

  • According to the video, if everyone has the key (media literacy), then the world would have more ______ ?
  • The video provided guide questions to understand any media message further:
  • Who created this message?
  • What creative techniques are used to attract my attention?
  • How might different people understand this message differently from me?
  • What lifestyles, values, and points of view are represented in, or omitted from, this message?
  • Why is this message being sent?
