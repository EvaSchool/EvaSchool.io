Distance vs Displacement

Distance:

  • Distance is a scalar quantity. (how to link I want this as like a callout when clicked? -> A scalar quantity is a measurement described solely by its magnitude without a specific direction.) 
  • It measures the total path traveled by an object, regardless of direction.
  • Distance is always positive or zero.
  • It provides information about the "how much" of a journey.
  • For instance, if you walk 3 kilometers north and then 4 kilometers south, the total distance covered is 7 kilometers.

Scenario: Imagine that you have a piece of string and you have tied it to your starting position and as you move, the string uncurls and is left behind you so it traces out how far you have walked. This is a measure of the total length of the string, with the base unit expressed in meters.

Displacement:

  • Displacement is a vector quantity. (how to link I want this as like a callout when clicked? -> A vector quantity is a physical measurement that has both magnitude and direction. ) 
  • It measures the change in position from the starting point to the ending point of an object's journey, including direction.
  • Displacement can be positive, negative, or zero, depending on whether the object ends up in a position that is ahead, behind, or at the starting point.
  • It provides information about "how far out of the way" an object has gone.
  • For example, if you walk 3 kilometers north and then 4 kilometers south, your displacement is 1 kilometer south because you've ended up 1 kilometer south of your starting point.

Scenario: The string is fixed at the start and uncoils as you walk (same as before) only now when you stop you pull on the rope until is is taught (so there is no slack between the start and end points) and this is the displacement. It is the straight line distance between the start and end points, with the base unit expressed in meters.
