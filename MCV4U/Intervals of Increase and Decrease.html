Determining the intervals in which a function increases and decreases is extremely useful for understanding the behavior of the function. The following statements give a clear picture. 

Intervals of Increase and Decrease 

We say that a function f is decreasing on an interval if, for any value of x₁ < x₂ on the interval, f(x₁) > f(x₂).

Similarly, we say that a function f is increasing on an interval if, for any value of x₁ < x₂ on the interval, f(x₁) < f(x₂).
