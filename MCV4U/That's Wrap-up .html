That's a Wrap

 

Learning Goals:

  • Students are familiar with the idea of the limit of a function as x approaches a certain value (“a”).
  • Students are familiar with the properties of limits.
  • Students can use the properties of limits to calculate the limits of polynomials.
  • Students know that there are a number of techniques for calculating limits, but in any case, a graph or table of values can help to check results.

Success Criteria 

  • I CAN….
    Explain the concept of the limit of a function as x approaches a certain value (“a”).
  • Identify the correct property when evaluating limits. 
  • Use the properties of limits to calculate the limits of functions.
  • Identify the techniques to use in calculating limits of functions and functions of indeterminate forms.
