Factors that Influence Supply

  1. Change in technology
  2. Changes in production and marketing costs
  3. Changes in taxation
  4. Changes in weather (e.g., frost can destroy a crop)
  5. Changes in the number of sellers (e.g., home videos were once widespread, but now people can download videos at home, so the need for video rental stores has decreased)

When there is a change in one of the factors that influence supply, the supply curve will shift.  This will be either an increase or decrease in supply. Suppose there is a decrease, the supply curve shifts to the left. Suppose there is an increase, the supply curve shifts to the right.

There is an essential distinction between a “change in supply” and a “change in quantity supplied.” The same rule applies here as with the demand curve. 

If the ONLY factor that changes is the price, then the result would be a movement along the supply curve from one quantity to another. The supply curve will not shift; instead, there will be a movement along the original supply curve. In this case, we would say there was a “change in quantity supplied.” If another factor (other than price) changes, then the supply curve would shift altogether, and this would be identified as a “change in supply.”
