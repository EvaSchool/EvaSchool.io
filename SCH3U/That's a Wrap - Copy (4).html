That's a Wrap

Learning Goals

In this lesson you should achieve the following learning goals:

  • Differentiate Qualitative and Quantitative Analysis
  • Understand the importance of accuracy in industrial chemical processes and its potential impact in the environment if quantitative accuracy is not observed
  • Learn the issues in chemical quantities
  • Learn about Mole 
  • Solve Mole problems
  • Learn about Molar Mass
  • Determine the molar mass of an element and a compound
  • Solve problems involving Molar masses
  • Learn about Mass and the number of Entities
  • Determine the number of entities
  • Understanding Combustion Analysis and Percentage Composition
  • Solve Problems involving Percentage Composition
