Using the Amount Concentration Formula

When using this formula always check that:

  • the equation is rearranged correctly to solve for the required variables
  • amount is expressed in mol
  • volume is expressed in L. 

The most common conversion you will encounter is mL to L and vice versa. (1 mL = 1 x 10-3 L e.g. 5.0 mL = 0.0050 L or 5.0 x 10-3 L)
